# Module2026
Решение всех ваших проблем

> [!WARNING]
> На экзамене присутствует внешняя комиссия. Использование скриптов автоматизации может быть расценено как нарушение правил и привести к отстранению от экзамена.
>
> **Используйте на свой страх и риск.** Обязательно изучите код перед использованием - вы должны понимать каждую настройку и уметь объяснить её вручную.

Набор Bash-скриптов для полной автоматизации развёртывания инфраструктуры в рамках демонстрационного экзамена. Скрипты запускаются на хосте Proxmox и управляют виртуальными машинами через `qm guest exec`.

> Скрипты покрывают задания **Модуля 1** и **Модуля 2**, чего достаточно для оценки «4».
